# Quiz Application - Microservices Architecture with Spring Boot

This project is a Quiz Application built using Spring Boot and designed with a microservices architecture. 
The application consists of three microservices, including an API Gateway for seamless request routing. The core features include:

  **Quiz Creation: Create quizzes dynamically with customizable questions.**
  **Question Management: Handle various types of questions (multiple-choice, true/false, etc.).**
  **Answer Evaluation: Calculate the number of correct answers and provide instant feedback.**

# Getting Started

  1. Clone the repository.
  2. Install the dependencies.
  3. Run the application.

# Usage

  1. Go to the Quiz application website.
  2. Create a new quiz.
  3. Take the quiz.
  4. View the results.
